# JS 运行机制

## 看题：
```js
  console.log(1);
  setTimeout(function () {
    console.log(2);
  }, 0);
  console.log(3);
  // 1 3 2

  console.log('A');
  while(1) {

  }
  console.log('B');
  // A while同步，永远不会执行到B（相当于一个死循环）、

  console.log('A');
  setTimeout(function () {
    console.log('B');
  }, 0);
  while(1) {

  }
  // A 同上

```
